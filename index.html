Research

Development of a high-fidelity failure prediction system for reinforced concrete bridge columns using generative adversarial networks

Ting-Yan Wu, Rih-Teng Wu, Ping-Hsiung Wang, Tzu-Kang Lin, Kuo-Chun Chang
Engineering Structures, 2023
Code / Paper

Conditional Generative Adversarial Network (CGAN)-base network is proposed to predict the failure of reinforced concrete bridge columns. The proposed cGANRC is able to forecast the surface damage patterns of the bridge columns according to given Aspect Ratio (AR), Longitudinal Reinforced Ratio (LR), Transverse Reinforced Ratio (TR) and damage index (DI).

Building Manufacturing Deep Learning Models with Minimal and Imbalanced Training Data Using Domain Adaptation and Data Augmentation

Adrian Shuai Li, Elisa Bertino, Rih-Teng Wu, Ting-Yan Wu,
IEEE International Conference on Industrial Technology (ICIT), 2023
Paper

In this paper, we propose a novel domain adaptation (DA) approach to address the problem of labeled training data scarcity for a target learning task by transferring knowledge gained from an existing source dataset used for a similar learning task. Our approach works for scenarios where the source dataset and the dataset available for the target learning task have same or different feature spaces.
